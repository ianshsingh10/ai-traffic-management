 
   # ParkAlert — Flipkart Gridlock 2.0

   AI-driven parking intelligence system. Detects illegal parking hotspots
   from violation data and quantifies their congestion impact to enable
   targeted enforcement.

   ## Setup

   ```bash
   pip install pandas numpy scikit-learn folium flask flask-cors
   cd frontend
   npm install
   cd ..
   ```

   ## Run order

   1. Put the raw violations CSV into `data/`
   2. `python backend/phase1_data_prep.py`   → cleans + scores data
   3. `python backend/phase2_hotspot_engine.py` → K-Means clusters + patrol schedule
   4. `python backend/phase3_heatmap.py`     → builds the two Folium maps
   5. Copy maps so the React app can load them:
      ```bash
      copy maps/*.html frontend/public/maps/
      ```
   6. Start the backend:
      ```bash
      cd backend
      python app.py
      ```
      Runs on http://localhost:5000
   7. In a second terminal, start the frontend:
      ```bash
      cd frontend
      npm start
      ```
      Opens http://localhost:3000

   ## Folder structure

   ```
   data/      cleaned CSVs + JSON outputs (generated by phase 1 & 2 scripts)
   backend/   Flask API + the 3 phase scripts
   maps/      generated Folium HTML maps (phase 3 output)
   frontend/  React dashboard
   ```

   ## API endpoints (backend/app.py)

   - `GET /api/summary`              — top-level KPIs
   - `GET /api/zones`                — police-station level ranking
   - `GET /api/hotspots`             — K-Means cluster profiles
   - `GET /api/schedule`             — AI patrol schedule
   - `GET /api/trends`               — hourly/monthly/vehicle breakdowns
   - `GET /api/violations`           — paginated raw records
